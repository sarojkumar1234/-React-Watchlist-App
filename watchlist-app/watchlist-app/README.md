# Watchlist App

A simple React app to view market instruments and manage your watchlist.

## Features

- View list of instruments with prices and percentage changes
- Search instruments by symbol
- Add/remove instruments to watchlist
- Sort watchlist by price
- Watchlist persists in browser storage

## Setup

```bash
npm install
npm run dev
```

Built with React, TypeScript, and Vite.
